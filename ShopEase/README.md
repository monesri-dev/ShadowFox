# 🛍️ ShopEase - Modern E-Commerce Website

ShopEase is a responsive front-end e-commerce website built using **HTML5, CSS3, and Vanilla JavaScript**. It demonstrates a complete online shopping experience including product browsing, product details, shopping cart, filtering, sorting, and Local Storage.

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/ShopEase.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

---

## 🌐 GitHub Pages Deployment

1. Push the project to GitHub.
2. Go to **Repository → Settings → Pages**.
3. Under **Source**, choose:
   - Branch: `main`
   - Folder: `/ (root)`
4. Click **Save**.
5. Wait a few minutes.
6. Your website will be live.
